Overview

Todo App Project is a full-stack task management application built using HTML, CSS, JavaScript, Node.js, Express, and MySQL. The project focuses on practicing real-world full-stack concepts such as authentication, authorization, RESTful APIs, and database-driven applications. It includes role-based access control, where normal users manage tasks and admin users access a protected admin dashboard.


Features:
User authentication: Sign up, Sign in, Logout.
Role-based access control (Admin & User).
Admin dashboard (protected route).
Create, update, and delete tasks.

Task status filtering:Completed tasks, In-progress tasks, Pending tasks.

Task priority filtering:High priority, Medium priority, Low priority.

Time-based task queries:Tasks posted, updated, or deleted within specific day ranges.

Dashboard statistics overview

Responsive sidebar navigation

Tech Stack

Frontend:HTML, CSS, JavaScript

Backend:Node.js, Express.js

Database:MySQL

Other Tools & Libraries:JWT authentication, bcrypt for password hashing, dotenv (environment variables), CORS, and more. 

Admin Access

Admin routes are protected using middleware

Only users with the admin role can access the admin dashboard

Unauthorized users are redirected automatically

Purpose of the Project

This project is a learning-focused full-stack application built to demonstrate practical backend and frontend integration, authentication, authorization, and database operations. It is not intended for production just to practice the full full stack development concept.
